# Urbi

> *To the city.* — Inward coherence, memory, dream, and self-audit engine.

Urbi is the inward half of the Urbi/Orbi duality.
It maintains the canonical internal state of the system —
handling memory consolidation, the dream/offline cycle (ΦΔ),
self-audit, causal modelling, and belief revision.
Where Orbi acts, Urbi reflects.
