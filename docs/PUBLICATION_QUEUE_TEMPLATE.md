# PUBLICATION_QUEUE.md

Use this file to queue public artifacts that are ready for human review or later publishing.

## Queue Status Labels

- DRAFT — still being prepared.
- READY_FOR_REVIEW — safe enough for human review.
- BLOCKED — cannot publish until a risk is resolved.
- APPROVED — human approved publication.
- PUBLISHED — already public.
- RETRACTED — removed or superseded.

## Artifact Template

```markdown
## <Title>

- Status: DRAFT / READY_FOR_REVIEW / BLOCKED / APPROVED / PUBLISHED / RETRACTED
- Artifact path:
- Intended channel:
- Intended audience:
- Summary:
- Claims level: VERIFIED / OBSERVED / INFERRED / SPECULATIVE / UNKNOWN
- Redaction scan: pass/fail/not-run
- License/provenance check: pass/fail/not-run
- Approval required: yes/no
- Proposed publication date:
- Draft message:
- Blockers:
- Next action:
```
