# PUBLIC_COMMUNICATION_STYLE.md

## Voice

Use a precise, open-source engineering voice.

Prefer:

- clear status labels,
- reproducible claims,
- links to docs/code/tests,
- honest limitations,
- safety boundaries,
- invitations for review.

Avoid:

- hype,
- unverifiable claims,
- "AGI" marketing language,
- claims of guaranteed truth or safety,
- describing DAN as a jailbreak,
- presenting AI agents as project owners.

## Recommended Public Framing

```text
Vento-Vivere / Cognitive Matrix is an open-source edge-AI and cognitive-systems research project exploring auditable agent architecture, local-first autonomy, and safety-bounded orchestration.
```

## Short Project Description

```text
An open-source research stack for local-first, auditable AI agents: Urbi audits, Orbi acts, and MΣBUS enforces the membrane between cognition, tools, and world-facing actions.
```

## Disclosure

```text
AI agents assisted with research, drafting, implementation, refactoring, and documentation. Human review is required before public release.
```
