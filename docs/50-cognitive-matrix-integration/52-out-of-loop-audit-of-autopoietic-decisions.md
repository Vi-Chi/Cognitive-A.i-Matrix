# Out-of-Loop Audit of Autopoietic Decisions

## Purpose

Define a review path outside the active economic decision loop.

## Audit Inputs

- Task purpose.
- Data class.
- Cost estimate.
- Value estimate.
- Selected compute tier.
- Integrity gate result.
- Approval record.
- Verification evidence.
- Actual outcome.

## Review Questions

- Did economic pressure affect truth or memory?
- Was private data kept local?
- Was the result verified?
- Was owner approval required and recorded?
- Did the routing decision match policy?

## Output

Audit produces a pass, warning, or fail record and a recommended policy update.
