# Integrity Gate for Economic Actions

## Purpose

Define the veto layer for spending, external compute, services, and treasury
changes.

## Gate Checks

- Truthfulness.
- User autonomy.
- Memory integrity.
- Privacy.
- Local sovereignty.
- Legality.
- Safety.
- Auditability.
- Verification.

## Outcome

The gate returns one of:

- approve for simulation,
- approve with owner review,
- refuse,
- defer until evidence improves.

## Rule

Treasury policy consumes the gate result. It does not override it.
