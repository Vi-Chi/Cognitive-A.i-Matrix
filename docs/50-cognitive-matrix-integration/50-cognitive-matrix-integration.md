# Cognitive Matrix Integration

## Purpose

Define how Autopoiesis supports the Cognitive Matrix without changing its
integrity core.

## Integration Flow

1. Cognitive Matrix proposes a compute need.
2. Autopoiesis classifies the task and data.
3. Integrity gate checks privacy, memory, safety, legality, and auditability.
4. Cost and value estimators run.
5. Treasury policy approves simulation, refusal, deferment, or human review.
6. SigmaBUS records provenance and decision evidence.

## Boundary

Autopoiesis can recommend a compute route. It cannot alter memory truth, audit
rules, or self-correction logic.
