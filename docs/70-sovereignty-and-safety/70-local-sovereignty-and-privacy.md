# Local Sovereignty and Privacy

## Purpose

State the data boundary for local-first operation.

## Local by Default

- Private Cognitive Matrix memory.
- Omni-AI logs.
- Credentials.
- Vessel and infrastructure details.
- Unpublished project files.
- Personal documents.
- Private RAG corpus.
- Local service endpoints.
- Sensitive telemetry.

## Allowed Public Data

- Sanitized metadata.
- Public service descriptions.
- Non-sensitive benchmark summaries.
- Public proof of completion.
- Provider capability records.

## Rule

When the data class is unclear, treat it as private and stay local.
