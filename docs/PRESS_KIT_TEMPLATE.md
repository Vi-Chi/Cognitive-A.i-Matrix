## Project

Vento-Vivere / Cognitive Matrix

## One-line Description

An open-source edge-AI research stack for auditable, local-first agent architecture with strict separation between audit, action, and routing.

## Longer Description

Vento-Vivere / Cognitive Matrix explores local-first AI systems where cognition, auditing, tool use, and world-facing actions are separated by enforceable protocol boundaries. The architecture uses Urbi as an audit/judgment layer, Orbi as the action/tool layer, and MΣBUS as the membrane that enforces routing, provenance, and safety gates.

Research and engineering work in progress. Public artifacts should distinguish verified implementation from active design and speculation.
