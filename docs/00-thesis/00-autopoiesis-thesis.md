# Autopoiesis Thesis

## Purpose

Define Project Autopoiesis as the economic survival layer around the Cognitive
Matrix.

## Context

Omni-AI runs operations. Cognitive Matrix handles integrity-aware cognition.
SigmaBUS carries semantic events and provenance. Autopoiesis asks what cognition
costs, how the cost can be budgeted, and when useful work can replenish that
budget.

## Thesis

A long-lived cognitive system should track the cost of its own cognition and
earn or conserve resources through useful, honest, auditable contribution.

## Non-Negotiable Boundary

Economic pressure cannot change truth, memory, audit, privacy, user autonomy, or
local-sovereignty rules. The integrity gate must run before treasury logic.

## Phase 0 Position

The project is a simulator and design repository. It does not spend funds,
deploy public services, export private memory, or write to chains.
