# ICP Role in Autopoiesis

## Candidate Role

ICP may host public coordination logic for a provider registry, job lifecycle
state, reputation metadata, and treasury simulation records.

## On-Chain Candidate Data

- Public provider capability summaries.
- Sanitized job metadata.
- Public benchmark summaries.
- Audit pointers that do not reveal private data.

## Off-Chain Data

- Private Cognitive Matrix memory.
- Private Omni-AI logs.
- Credentials and wallet material.
- Raw local service endpoints.
- Private job inputs and outputs.

## v0 Scope

Start with local JSON schemas and a simulator. Treat canisters as a later target
once schema behavior and privacy boundaries are stable.
