# Storage and Public Memory Options

## Candidate Role

Filecoin, Arweave, and similar storage systems may hold public artifacts,
published releases, benchmark proofs, or durable research outputs.

## Must Stay Local

- Private memory.
- Personal files.
- Unpublished project notes.
- Credentials.
- Raw telemetry.
- Private RAG corpus data.

## Rule

Use public storage only for public data with a reason, a retention plan, and an
owner-approved publication boundary.
