# Community Post Templates

These are reusable starting points. Rewrite for each community after checking local rules.

## Short Social Update

```text
I’m building Vento-Vivere / Cognitive Matrix: an open-source local-first AI research stack for auditable agents and safety-bounded orchestration.

Current focus: <specific milestone>.

What I’m looking for: feedback from <audience> on <specific question>.

Link: <public link>

Disclosure: I’m the originator/maintainer; AI tools assisted with research/drafting/implementation.
```

## Technical Forum Review Request

```text
Title: Request for review: <specific technical topic>

I’m working on an open-source local-first agent architecture where one layer audits, one layer acts, and a membrane enforces routing/provenance/action gates.

Current status: <verified/observed status>.

Specific review request: <one concrete question>.

What is already available: <repo/docs/demo/tests>.

Known limitations: <limitations>.

Disclosure: I’m the originator/maintainer of the project. AI tools assisted with drafting and implementation; I’m looking for technical critique, not promotion.
```

## Discord / Matrix Update

```text
Small project update/request: I’m building an open-source local-first AI/agent stack focused on auditable actions and explicit safety gates.

I’m currently working on <specific topic>. Does anyone here have experience with <specific technology/question>?

I can share the repo/docs if appropriate for this channel. Disclosure: I’m the maintainer.
```

## Build Log / Maker Post

```text
Build log: <hardware/software milestone>

Goal: <practical goal>.
Hardware/software: <versions>.
What worked: <observed facts>.
What failed: <failure cases>.
Safety boundaries: <what is not live/automated>.
Next test: <next reproducible step>.

Feedback wanted: <specific ask>.
```

## Response to Criticism

```text
Thanks — that is a valid concern. Current status is <verified/observed/inferred>, not a production claim. I’ll capture this as <issue/doc update/test need> and avoid presenting it as solved until it is measured or reviewed.
```
