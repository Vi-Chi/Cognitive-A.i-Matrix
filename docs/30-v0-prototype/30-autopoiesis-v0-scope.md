# Autopoiesis v0 Scope

## Purpose

Define the smallest useful prototype.

## v0 Components

- Provider registry.
- Treasury simulator.
- Compute decision router.
- Job lifecycle manager.
- Audit ledger.
- Integrity gate.

## Storage

Start with local JSON files or SQLite. Chain integration waits until local
schema behavior is tested.

## No-Go Items

- No real spending.
- No private-memory export.
- No public service deployment.
- No chain writes.
- No autonomous treasury policy changes.
