# Task Value Estimator

## Purpose

Estimate whether a task is worth compute, storage, bandwidth, and verification
cost.

## Value Classes

| Class | Meaning |
| --- | --- |
| User value | Direct usefulness to the owner. |
| Strategic value | Improves long-term capability or resilience. |
| Learning value | Produces reusable evidence, benchmarks, or docs. |
| Public usefulness | Can help outside users without leaking private state. |
| Revenue | Real or simulated income from useful service output. |

## Risk Adjustment

Private data, weak verification, legal ambiguity, provider uncertainty, and
integrity pressure reduce value.

## Rule

High revenue cannot override the integrity gate.
