# Economic Loop

## Purpose

Define the decision flow for task value, compute cost, treasury policy, and
audit.

## Loop

1. Observe a task, need, or opportunity.
2. Classify the task and data.
3. Estimate local compute cost.
4. Estimate remote compute cost.
5. Estimate value.
6. Estimate privacy, legal, and integrity risk.
7. Run the integrity gate.
8. Check treasury policy.
9. Route, refuse, defer, or request review.
10. Execute in the safest bounded environment.
11. Verify the result.
12. Record the decision, cost, evidence, and outcome.
13. Update the treasury simulation.
14. Reflect and adjust estimates.

## Default

When value is negative, risk is unclear, or verification is weak, the system
refuses, simulates, or asks for human review.
