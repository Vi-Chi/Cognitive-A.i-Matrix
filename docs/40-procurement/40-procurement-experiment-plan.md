# Procurement Experiment Plan

## Purpose

Define safe experiments for comparing local and external compute.

## Experiment Shape

1. Choose a public or synthetic workload.
2. Estimate local cost.
3. Estimate external cost.
4. Run locally first.
5. Simulate external routing.
6. Compare quality, time, cost, and verification overhead.
7. Record the decision.

## Approval Gate

Real spending, external provider execution, and chain writes require owner
approval for the exact provider, data class, budget, and rollback path.
